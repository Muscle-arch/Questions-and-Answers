# 基于大模型与RAG技术的四川大学校内问题解答网站

**王崇宇** 2023141461054　　**刘喻尧** 2023141461040　　**程瑞鑫** 2023141461108

---

## 一、项目名称

基于大模型与RAG技术的四川大学校内问题解答网站

## 二、开发工具

- 代码编辑器：VS Code

## 三、开发语言

- 前端：Vue
- 后端：Python

## 四、项目性质

- 联网运行
- 多用户同时访问
- 支持智能问答 + 校内导航双功能

## 五、项目功能概述

### 1. 智能问答

| 步骤           | 说明                                     |
| -------------- | ---------------------------------------- |
| 用户访问       | 用户通过浏览器打开网址即可使用           |
| 提问交互       | 提供聊天式界面，用户输入问题             |
| RAG 知识库检索 | 系统根据用户问题，从知识库中查找相关资料 |
| 构建 Prompt    | 将资料 + 问题 + 要求组合，发给 AI        |
| AI 回答生成    | 调用云端大模型 API，生成准确回答         |

### 2. 校内导航

| 步骤          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 导航识别      | 自动识别"位置查询、路线规划"类指令，提取起点、终点 |
| 地图 API 调用 | 对接地图开放平台 API，生成校内步行最优路线         |
| 结果展示      | 同步返回可视化导航地图                             |

## 六、系统架构

| 模块          | 职责                                                                                    |
| ------------- | --------------------------------------------------------------------------------------- |
| 前端          | 对话展示、输入交互；集成地图组件，实现校内路线可视化展示                                |
| 后端          | 接收请求、RAG 检索、Prompt 拼接、AI 接口调用；导航指令解析、地图 API 调用、路线数据处理 |
| 知识库        | 存储文档、资料、题库；校内地标信息库                                                    |
| AI 接口       | 提供语义理解、问答生成                                                                  |
| 地图导航 API  | 提供校内定位、路线规划、路径计算服务                                                    |
| 服务器 & 域名 | 支持网站联网部署、多用户稳定访问                                                        |
